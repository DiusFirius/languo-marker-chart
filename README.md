# Languo 240 Marker Color Chart

Interactive web application for browsing and searching Languo acrylic marker colors.

## Features
- 240 color swatches in exact physical chart layout
- Instant search by marker ID (partial matching supported)
- Responsive design (maintains 20Ã—12 grid on all devices)
- Continuous blink animation for found markers

## Usage
1. Visit: [your-github-pages-url]
2. Browse the color grid (matches physical chart layout)
3. Search for markers using the search box (e.g., "RY" or "RY-01")
4. Found markers will blink until you clear the search

## Data Updates
To update marker colors or IDs:
1. Edit `js/markers-data.js`
2. Commit and push to main branch
3. Changes deploy automatically within 1-2 minutes

## Development
- Pure HTML/CSS/JavaScript
- No build process required
- Host anywhere static files are supported

## Note
Color accuracy on screen is approximate. Actual marker colors may vary. For professional color matching, refer to physical markers.
